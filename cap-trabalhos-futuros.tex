\chapter{Trabalhos Futuros}
\label{cap:trabalhos_futuros}

Mesmo atentendo ao objetivo proposto, existem features (recursos) e upgrades
(atualizações) de melhorias do projeto atual, como por exemplo, a representação
do resultado em um gráfico mais complexo. A seguir serão listadas todas as
alterações identificadas como melhorias para implementações futuras:

\begin{itemize}
  \item Exibir o total de respostas e média para cada pergunta da
  avaliação.
  \item Permitir a adição e alteração de perguntas durante a avaliação de uma
  aula. 
  \item Utilização de um visual mais sofisticado para aplicativos Android, como
  por exemplo o conceito do Material Design~\cite{material_design}.
  \item Utilização de bibliotecas mais utilizadas pela comunidade Android como o
  Retrofit~\cite{retrofit}, EventBus~\cite{eventbus}.
  \item Adição de push notification~\cite{push_notification} para informar aos
  alunos quando tiver uma nova avaliação sem a necessidade de abrir o
  aplicativo.
  \item Paginação durante a busca de avaliações, disciplinas, matérias e aulas
  para serem avaliadas.
  \item Cache para funcionamento offline.
  \item Permitir ao professor a criação de avaliações de cada aula.
\end{itemize}
